# Jarvis Lite (MVP)
1) Crée/liste/cherche des notes
2) Crée/liste des rappels (tick toutes 15s)
3) Lance des programmes/fichiers
+ Humour intégré, TTS optionnel

### Setup
```bash
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env  # renseigne OPENAI_API_KEY si tu veux évoluer plus tard
python main.py
```
